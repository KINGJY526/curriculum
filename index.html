디자인컨버전스학부의 교육과정은 핵심을 구성하는 Core 과목들을 따라 학부교육의 너비와 깊이를 형성하는 Electives 과목들을 통하여 구체화된다. 3학년 Core과목에는 Node 과목들이 속하여 전문성을 기를 수 있는 과정을 제공한다.

01

Core

기본적 표현 능력을 갖춘 뒤 발상과 융합 능력을 기르는 스튜디오 교과 중심으로 전개됩니다.

02

Node

3학년 때 하나의 과정을 선택해 통년으로 이수하며, 분야별 전문성을 심화합니다.

03

Elective

디자인과 사회·문화·기술적 이슈를 다루는 다양한 선택 교과들을 제공합니다.
